# 現行仕様（2026-02-15時点）

## 画面構成

- 上部固定ヘッダー
  - `メニューを開く`
  - `来店者を管理`
- メモ画面
  - 注文一覧（件数、合計杯数）
  - 各行に数量ステッパー（`- / 数量 / +`）と注文者選択
- 下部固定フッター
  - `確認`
  - `メモをリセット`
- モーダル
  - メニュー選択
  - 来店者管理
  - 確定オーダー確認

## 注文フロー

- `メニューを開く` -> 商品タップで注文追加
- 追加時は `個数1 / 注文者未選択`
- 追加後もメニューモーダルは開いたまま（連続追加可能）
- 追加時に「`◯◯ を追加しました`」通知を短時間表示する

## 数量仕様

- 範囲: `1〜99`
- `-` で 1 未満にならない
- `+` で 99 を超えない
- `+` 長押し
  - 開始: 300ms
  - 間隔: 100ms
  - 停止: pointer up / cancel / leave

## 来店者仕様

- 来店者モーダルで追加・削除
- 重複名は追加不可
- 注文者選択肢は `未選択 + 来店者一覧`
- 来店者一覧はクッキー（`bar_visitors`）に保存する
- ページ再読込後も来店者一覧を復元する（保存期間: 約1年）

## 確認仕様

- `確認` 押下で確定オーダーモーダルを表示
- 確定オーダーは最新の確認内容で上書き
- `閉じる` / `通常表示へ戻る` は表示切り替えのみ（メモ保持）

## リセット仕様

- `メモをリセット` 押下時に確認ダイアログ
- OK のときのみメモをクリア

## 表示仕様

- メモ/確定オーダーともに価格・時刻は表示しない
- 注文行は `商品名 / 杯数 / 注文者` を表示
