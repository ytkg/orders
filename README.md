# orders

Issue #1 の初期実装として、バーの注文メモ画面を追加しました。

## セットアップ

```bash
npm install
npm run dev
```

## テスト

```bash
npm run test
npm run test:watch
```

## 実装済み機能

- UIライブラリとして Material UI（MUI）を導入
- 注文メモの追加（お酒の種類 / 個数 / 注文者）
  - お酒はメニューモーダルを開いて選択し、タップで即追加（モーダルは閉じない）
  - 追加直後は「個数1 / 注文者未選択」
  - 追加時に短時間の通知を表示
  - 個数と注文者はメモ一覧側で選択して編集
  - フッターの「確認」で確定オーダービューを表示
- 来店者（来店グループ）の登録・削除（必要時に管理ポップアップを開く）
- 入力バリデーション（必須項目、個数の正数チェック）
- 追加したメモの一覧表示
- メモの削除、注文者の再選択、明示的リセット
